Returns various data that is provided to the algorithm for the left sleeper. Each value is separated by a space:

`"<Birth Year> <Height> <Weight> <Bed> <Person> <A2D> <Chamber> <Wedge>"`


# Birth Year #

Sleeper's Birth Year


# Height #

Sleeper's height in inches 


# Weight #

Sleeper's weight in pounds


# Bed # 

Bed Type 

| Value | Description |
| ---- | ---- |
| 0 | Unknown | 
| 1 | Twin |
| 2 | Full |
| 3 | Queen |
| 4 | King |


# Person #

Type of person 
| Value | Description |
| ---- | ---- |
| 0 | Adult | 
| 1 | Child |
| 2 | Unknown |


# A2D #

A2D Mode

| Value | Description |
| ---- | ---- |
| 0 | Low | 
| 1 | High |
| 2 | Split |
| 3 | Unknown |


# Chamber #

Type of chamber

| Value | Description |
| ---- | ---- |
| 0 | Normal | 
| 1 | Kids |
| 2 | Wedge |
| 3 | Unknown |


# Wedge #

Wedge head angle

| Value | Description |
| ---- | ---- |
| 0 | Normal | 
| 1 | Wedge |
| 2 | Invalid |
