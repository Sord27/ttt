Return four numbers, separated by spaces, e.g. `"0 0 0 0"`

Each of these numbers tell how many reboots have occurred due to various issues.

`"<1> <2> <3> <4>"`

| Number | Reboot Issue |
| ---- | ---- |
| 1 | Failure in USB Receiving |
| 2 | Failure in USB transmitting |
| 3 | Task faults |
| 4 | Full Heap |
