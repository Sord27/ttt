Gets the front LEDs current pattern. Will return a value from the table below.

| Pattern | Value | SP1 | SP2 |
| ---- | ---- | ---- | ---- |
| Disabled | 0 | Y | Y |
| Tetris | 1 | Y | Y |
| Knight-Rider | 2 | Y | Y |
| Cross | 3 | Y | Y |
| Flashing | 4 | Y | Y |
| Fast Flashing | 5 | N | Y |
| Slow Flashing | 6 | N | Y |
| All On | 7 | N | Y |
| All Off | 8 | N | Y |
